Backup Database

Proses Unduh ini akan mengunduh keseluruhan database PBB anda, format file (.sql)

Unduh Database

  • Usahakan untuk melakukan backup secara rutin dan terjadwal.
  • Backup yang dihasilkan sebaiknya disimpan di komputer terpisah dari server yang digunakan.

Restore Database

Backup yang dibuat dapat dipergunakan untuk mengembalikan database PBB anda apabila ada masalah. Klik tombol Restore di bawah untuk menggantikan keseluruhan database PBB dengan data hasil backup terdahulu.

Proses ini akan membutuhkan waktu beberapa menit, menyesuaikan dengan spesifikasi komputer server PBB dan sambungan internet yang tersedia.
